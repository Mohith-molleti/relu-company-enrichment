# AI Company Enrichment Dashboard

## Overview

AI Company Enrichment Dashboard is a Streamlit application that extracts company intelligence from websites using Gemini AI.

The application:
- Scrapes company websites
- Extracts contact information
- Identifies core services
- Finds target customers
- Generates probable pain points
- Creates outreach openers

## Features

- Multi-company URL processing
- AI-powered company profiling
- Contact discovery
- CSV export
- JSON export
- Streamlit web interface

## Tech Stack

- Python
- Streamlit
- BeautifulSoup
- Pandas
- Gemini AI API
